# AI & Quantitative Finance Projects

This repository contains a collection of academic projects focused on
artificial intelligence applications in finance, trading, and natural
language processing.

The projects are designed for graduate-level study and research-oriented
applications.

---

## Projects Overview

### 1. LSTM Price Forecasting
Forecasting Bitcoin prices using Long Short-Term Memory (LSTM) neural networks.
- Time-series preprocessing
- Deep learning regression
- Financial forecasting

`01_lstm_price_forecasting`

---

### 2. Algorithmic Trading & Backtesting
Implementation and evaluation of quantitative trading strategies.
- SMA crossover strategies
- LSTM-based trading strategy
- ARIMA-based forecasting signals
- Backtesting using historical data

`02_algorithmic_trading`

---

### 3. NLP Sentiment Analysis
Binary sentiment classification on movie reviews using Recurrent Neural Networks.
- IMDB dataset
- Embedding + RNN
- Text preprocessing and inference

`03_nlp_sentiment_analysis`

---
